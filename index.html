Setup Configuration & Initialization
git config
Set or get Git configuration values (username, email, editor, etc.). Use --global to apply across all repos.
examplegit config --global user.name "John Doe" git config --global user.email "john@example.com" git config --list
--global --local --system --list
git init
Initialize a new, empty Git repository in the current directory. Creates a hidden .git folder.
examplemkdir my-project && cd my-project git init # → Initialized empty Git repository in .git/
git clone
Copy an existing repository (local or remote) into a new directory, including all history and branches.
examplegit clone https://github.com/user/repo.git git clone https://github.com/user/repo.git my-folder git clone --depth 1 https://github.com/user/repo.git
--depth --branch --bare
Basic Staging, Committing & Status
git status
Show the state of the working directory and staging area. Lists tracked/untracked, modified, and staged files.
examplegit status git status -s # short/compact format
-s--short-b
git add
Stage file changes to be included in the next commit. Use . to stage everything or specify individual files.
examplegit add index.html # stage one file git add . # stage all changes git add src/ # stage a folder git add -p # interactively pick hunks
-A-p--all
git commit
Save staged changes to the repository with a descriptive message. Each commit is a snapshot in history.
examplegit commit -m "Add login page" git commit -am "Fix typo" # stage + commit tracked files git commit --amend -m "Updated message"
-m-a--amend--no-edit
git diff
Show differences between working directory, staged area, and commits. Helps you see exactly what changed.
examplegit diff # unstaged changes git diff --staged # staged vs last commit git diff main feature-branch
--staged--stat--name-only
git rm
Remove files from the working tree and from the index. Stages the deletion for the next commit.
examplegit rm old-file.txt git rm --cached secret.env # untrack but keep file
--cached-r-f
git mv
Rename or move a file while keeping its Git history intact. Equivalent to mv + git add + git rm.
examplegit mv old-name.js new-name.js git mv src/utils.js lib/utils.js
Branching Branch & Merge Operations
git branch
List, create, rename, or delete branches. The starred branch is your current one.
examplegit branch # list local branches git branch feature/login # create new branch git branch -d feature/login # delete merged branch git branch -D feature/login # force delete git branch -m old-name new-name
-a-d-D-m-r
git checkout
Switch branches or restore working tree files. Use -b to create and switch in one step.
examplegit checkout main git checkout -b feature/nav # create + switch git checkout -- index.html # discard file changes
-b---f
git switch
Modern replacement for git checkout focused on branch switching. Cleaner and more intuitive.
examplegit switch main git switch -c feature/dashboard # create + switch git switch - # switch to previous branch
-c--detach-
git merge
Integrate changes from one branch into another. Creates a merge commit unless fast-forwarded.
examplegit checkout main git merge feature/login git merge --no-ff feature/nav # always create merge commit git merge --abort # cancel an in-progress merge
--no-ff--squash--abort
git rebase
Reapply commits on top of another base. Creates a cleaner, linear history compared to merging.
examplegit rebase main # rebase current branch onto main git rebase -i HEAD~3 # interactive: squash/edit last 3 commits git rebase --abort # cancel rebase
-i--onto--abort--continue
git stash
Temporarily save uncommitted changes to a stack so you can switch context without losing work.
examplegit stash # stash current work git stash pop # apply + remove top stash git stash list # see all stashes git stash apply stash@{1} # apply specific stash
poplistdropapply-u
Remote Remote Repository Operations
git remote
Manage connections to remote repositories. Add, remove, rename, or inspect remote URLs.
examplegit remote -v # list remotes with URLs git remote add origin https://github.com/user/repo.git git remote remove origin git remote rename origin upstream
-vaddremoverename
git fetch
Download changes from a remote without merging into your working branch. Safe way to inspect remote changes.
examplegit fetch origin git fetch --all # fetch all remotes git fetch origin main # fetch specific branch
--all--prune--tags
git pull
Fetch and immediately merge changes from the remote. Equivalent to git fetch + git merge.
examplegit pull git pull origin main git pull --rebase origin main # rebase instead of merge
--rebase--no-ff--ff-only
git push
Upload local commits to a remote repository. Use -u to set upstream tracking for a branch.
examplegit push origin main git push -u origin feature/nav # set upstream git push --force-with-lease # safer force push git push origin --delete old-branch
-u--force--force-with-lease--tags
History Viewing & Searching History
git log
Show commit history. One of the most powerful commands — use flags to customize the output format.
examplegit log git log --oneline --graph --decorate git log --author="John" --since="2 weeks ago" git log -p index.html # show patch for a file
--oneline--graph--decorate--author-p--stat
git show
Display detailed information about a specific commit: metadata, diff, and changed files.
examplegit show # show last commit git show a3f1b2c # show specific commit git show HEAD~2 # show 2 commits ago
--stat--name-only
git blame
Show who last modified each line of a file, with the commit hash and timestamp. Great for debugging.
examplegit blame index.html git blame -L 10,25 app.js # blame specific line range
-L-e-w
git grep
Search for a pattern (string or regex) through tracked files in the repository, like grep but Git-aware.
examplegit grep "TODO" git grep -n "console.log" # show line numbers git grep "api_key" HEAD~5 # search in older commit
-n-i-l--count
git tag
Mark specific commits as important milestones (like v1.0.0). Annotated tags include a message.
examplegit tag v1.0.0 git tag -a v2.0.0 -m "Major release" git tag -l # list all tags git push origin --tags # push tags to remote
-a-m-l-d
Undo Undoing & Fixing Mistakes
git restore
Discard changes in the working directory or unstage files. Modern replacement for parts of git checkout.
examplegit restore index.html # discard working dir changes git restore --staged app.js # unstage a file git restore --source=HEAD~2 config.js
--staged--source--worktree
git revert
Create a new commit that undoes the changes of a specific commit. Safe — doesn't rewrite history.
examplegit revert a3f1b2c git revert HEAD # revert last commit git revert HEAD~3..HEAD # revert last 3 commits
--no-commit-m
git reset
Move the branch pointer back to an earlier commit. Three modes: --soft, --mixed, --hard.
examplegit reset --soft HEAD~1 # undo commit, keep staged git reset HEAD~1 # undo commit, unstage git reset --hard HEAD~1 # undo commit + discard changes ⚠️
--soft--mixed--hard
git clean
Remove untracked files and directories from the working tree. Use -n first to preview what will be deleted.
examplegit clean -n # dry run – preview git clean -f # force delete untracked files git clean -fd # also remove untracked dirs
-n-f-d-x
Advanced Power User Commands
git cherry-pick
Apply a specific commit from one branch onto the current branch. Useful for grabbing a single fix without merging all changes.
examplegit cherry-pick a3f1b2c git cherry-pick a3f..b5c # range of commits git cherry-pick --no-commit a3f1b2c
--no-commit-x--abort
git bisect
Use binary search to find which commit introduced a bug. Git checks out commits until the bad one is found.
examplegit bisect start git bisect bad # current commit is broken git bisect good v1.0.0 # v1.0.0 was working # → test, then: git bisect good / bad git bisect reset # when done
startgoodbadreset
git reflog
Show a log of all HEAD movements — even from resets, rebases, checkouts. Your safety net to recover lost commits.
examplegit reflog git reflog show feature/nav # find the SHA and restore: git checkout -b recovered abc1234
git worktree
Check out multiple branches simultaneously in different directories from a single repository.
examplegit worktree add ../hotfix hotfix/login git worktree list git worktree remove ../hotfix
addlistremove
git submodule
Embed another Git repository inside your repo as a sub-project at a fixed commit.
examplegit submodule add https://github.com/user/lib.git git submodule update --init --recursive git submodule foreach git pull origin main
addupdateforeachstatus
git archive
Export a snapshot of the repository as a tar or zip archive without the .git history.
examplegit archive --format=zip HEAD > release.zip git archive --format=tar.gz main > project.tar.gz
--format--prefix--output
